Get started

Usage

01
Configure your provider

Open Settings, add your AI provider and API key, then click Done.

02
Connect a model

Gemini, OpenAI, Ollama, or any supported backend.

03
Open a project

Use /new to open a project folder.

04
Describe what to build

The agent generates files, edits code, and runs commands.

05
Go multi-session

Press F10 to toggle split-screen mode.

Changelog

Updates & Improvements

  • ▹Performance Optimization — Resolved performance lags; the application is now stable.
  • ▹New Settings Window — Introduced a dedicated Settings window with a significantly improved and cleaner interface.
  • ▹RTL Language Support — Fixed Arabic text rendering issues; text now correctly aligns to the right.
  • ▹Refined UI — Removed borders from AI messages for a cleaner, modern, elegant look.
  • ▹Easter Egg — Head over to Settings and click on the "Bowow" name to see a little surprise!
  • ▹Persistence — Chat sessions are now persistent. Your conversations are saved automatically.
Capabilities

Features

  • ▹Multi-Model Support — Works with Gemini, OpenAI, OpenRouter, Ollama, and llama.cpp backends.
  • ▹Split-Screen Mode — Toggle a 4-pane view (F10) to manage multiple build sessions simultaneously.
  • ▹Live File Editing — The agent reads, writes, and diffs project files directly on disk.
  • ▹Checkpoint System — Undo file changes with checkpoint-based rollback.
  • ▹Context Management — Automatic conversation compaction and pruning.
  • ▹Error Auto-Retry — Detects transient errors and retries with exponential backoff.
  • ▹MCP Tools — Integration of MCP Tools is now available (Experimental).
  • ▹System Prompt — Configure a custom system prompt to define AI behavior.
  • ▹User Prompts — Create, save, and manage multiple user prompts simultaneously.
  • ▹Terminal Commands — Enhanced terminal control and restrictions.
  • ▹Auto-Update System — Stay up-to-date effortlessly via the Settings menu.
